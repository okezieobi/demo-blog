[![Windows Build][appveyor-image]][appveyor-url]


Install dependencies:

```bash
npm install
```

Build the app:

```bash
npm run build
```


Start app in production mode at `http://localhost:5000/`:

```bash
npm start
```


Start app in development at `http://localhost:5000/`:

```bash
npm run dev
```
